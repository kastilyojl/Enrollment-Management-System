System Functions

The system has 5 types of users: Super Admin, Accounting, Registrar, Professor, and Student. Each user has specific functionalities:


Super Admin

- Full access to the system.

- Can create admin accounts.

- Can add subjects and courses (e.g., IT, Tourism).

- Can set academic dates.

Accounting

- Access limited to the Accounting dashboard.

- Can view payment history.

- Can verify and monitor student payments.

Registrar

- Can check student applications and approve or reject them.

- Can view the student list.
  
- Can set student IDs and distribute them.
  
- Can upload professor CSV files for grade submissions.

- Can upload clearance forms for students to download.

- Can monitor student clearance status.

Professor

- Can submit grades via the portal.

- Can submit reports for disciplinary actions, which can be approved by the Registrar.

Student

- Can submit their application and payment receipt.

- Can view their grades online.

- Can monitor the progress of their application and approval status.

- Multi-role access control

- Administrative dashboards for Accounting, Registrar, and Professor
